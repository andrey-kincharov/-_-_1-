Данный репозиторий создан для того чтобы показать мое решение тестового задания.

Текст тестового задания:
Следует самостоятельно создать обработку с табличной частью.
На форме обработки должны быть расположены элементы, привязанные к табличной части и кнопки, вызывающие упомянутые процедуры.
6 тестовых задач, задачу №5 обязательно решить с помощью запроса:
1. Создать обработку и процедуру заполнения её табличной части. Количество строк и длина в символах каждой строки должны задаваться на форме.
2. Подсвечивать строки в заполненной таблице, содержащие указанную в отдельном поле формы комбинацию символов.

3. Создать процедуру обратного отражения порядка строк (сортировки по номеру строки в обратном порядке.

4. Создать процедуру обратного отражения символов в каждой строке.
Например: строка была eab790 стала 097bae

5. Создать процедуру поиска дублей по заполненным значениям:
либо с перечислением номеров строк попарно(При этом номера строк не должны повторяться), например:
Значение eab790 в строке 1023 дублируется в строке 6740
Значение eab790 в строке 1023 дублируется в строке 2000
либо с перечислениями значений в дублируемых строках, например
Значение eab790 в строке 1023 дублируется в строках 2000, 6740.
Сообщать пользователю затраченное время в секундах, например:
Длительность поиска дублей 0.123 сек
Требуется решение данного пункта несколькими методами(минимум 2 варианта), в том числе запросом, с сообщением затраченного времени по каждому методу.
6. Создать процедуру удаления дублей строк
Дублями считаются те значения, что расположены ближе к концу таблицы.
Пример:
Было:
Номер строки|Значение
1 А
2 И
3 А
Стало:
Номер строки|Значение
1 А
2 И
